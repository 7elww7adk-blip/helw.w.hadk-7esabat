# نظام إدارة الفروع والتسويات - Laravel 11 (RTL)

هذا المشروع يقدّم **نظام إدارة أعمال متعدد الفروع** (وليس POS) لإدارة:
- الفروع الرئيسية والفرعية
- اليومية المحاسبية والمشتريات والمصروفات
- الخزنة والتحويلات الداخلية
- الجرد بالتقييم المالي
- تتبع رأس المال
- إدارة التوصيل والرحلات والتسويات
- التقارير الموحدة

## التقنية
- Laravel 11 / PHP 8.2+
- MySQL
- Blade + Tailwind
- Laravel Breeze (مصادقة)
- Chart.js

## خطوات التشغيل
```bash
cp .env.example .env
composer install
php artisan key:generate
php artisan migrate --seed
php artisan serve
```

## الأدوار
- Owner
- Manager
- Accountant
- DeliveryAgent
- Viewer

## قواعد أساسية
1) التحويلات بين الفروع لا تُحتسب مبيعات.
2) المندوب يُسلم كامل التحصيل، وأجره محسوب منفصلًا.
3) المخزون بالتقييم المالي فقط.
4) كل سجل مربوط بفرع.

## البيانات التجريبية
Seeder يضيف:
- فرع رئيسي + فرع فرعي
- مستخدمين حسب الأدوار
- إعدادات افتتاح
- يوميات ومشتريات ومصروفات
- حركات خزنة
- جرد
- تحويلات
- 3 مندوبين
- طلبات متعددة
- رحلة مفتوحة + رحلة مسوّاة + حالة عجز
