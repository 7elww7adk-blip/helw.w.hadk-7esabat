# 8) شرح تشغيل النظام خطوة بخطوة

1. ادخل على صفحة `index.html` بعد النشر.
2. سجل دخول بصاحب المحل.
3. من **الإعدادات** اضبط:
   - أنواع المصروفات
   - حالات الأوردرات
   - رسوم التوصيل الافتراضية
4. من **الفروع** تأكد من وجود:
   - الفرع الأساسي
   - الفرع الجديد
5. من **اليومية المحاسبية** سجّل بيانات اليوم لكل فرع.
6. من **التحويلات** سجّل أي تحويل داخلي (بدون تأثير مبيعات/مصروفات).
7. من **الدليفري**:
   - أضف أوردرات
   - اسندها لمندوب
   - راجع التسويات
8. من **التقارير** اعرض التقرير المطلوب حسب الشيت.

## 9) اقتراحات تطوير لاحقة بعد MVP
- دعم Attachments لفواتير/إيصالات Google Drive.
- دعم Print PDF للتقارير.
- فلاتر تقارير متقدمة (تاريخ/فرع/مندوب).
- إشعارات WhatsApp أو Telegram للأرصدة والتنبيهات.
- تحسين الأمان (2FA + تشفير أقوى + صلاحيات تفصيلية per action).
- ترحيل تدريجي إلى قاعدة بيانات احترافية (Cloud SQL/Firestore) عند التوسع.
